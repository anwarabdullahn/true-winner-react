### Installation

This Repo requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies and start the server.

```sh
$ git clone "this-repo"
$ cd "this-repo"
$ npm install || yarn install
$ npm run start || yarn start
```
